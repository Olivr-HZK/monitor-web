# 监测汇总网站说明

本文档介绍**监测汇总平台**网站的功能与界面，帮助您快速了解网站能做什么、页面上有哪些内容、以及如何使用。

---

## 一、网站是做什么的

监测汇总平台把多类监测内容集中在一个网站里展示，包括：

- **AI 热点**：AI 领域的新动态、技术突破、行业热点
- **热点趋势**：全网话题热度、讨论量、趋势变化
- **竞品社媒**：竞品在社交媒体上的动态与周报
- **休闲游戏**：各平台新游戏周榜、新玩法、竞品社媒与 UA 素材

您可以通过顶部导航和右侧「监测源」切换不同类型，在中间主区域浏览列表、点进某条看详情，或在休闲游戏下进入排行榜查看各平台游戏排名。

---

## 二、主要功能

### 1. 四种内容入口（顶部导航）

打开网站后，顶部有四个入口：

| 入口 | 作用 |
|------|------|
| **监测汇总** | 查看所有类型的监测内容，不区分类型 |
| **AI 热点** | 只看 AI 热点相关 |
| **趋势检测** | 只看热点趋势相关 |
| **休闲游戏检测** | 只看休闲游戏相关（新游戏、新玩法、竞品等） |

点击任一入口，中间列表和右侧监测源会一起切换，只显示该类型下的内容。

### 2. 休闲游戏下的细分

选择「休闲游戏检测」后，右侧边栏会出现更细的分类：

- **新游戏**  
  - 可以按平台选：**微信**、**抖音**、**iOS**、**安卓**，看该平台的小游戏/游戏周报。  
  - 主区域上方有 **「进入排行榜」** 按钮，点进去可以看各平台的游戏排行榜表格。
- **新玩法**  
  - 展示与「新玩法」相关的监测内容。
- **竞品**  
  - **社媒更新**：竞品社媒周报，还可以按**公司**筛选（公司列表来自周报）。  
  - **UA 素材**：竞品 UA 素材相关监测。

### 3. 列表的筛选和排序

主区域上方有一排筛选条件：

- **时间范围**：过去 1 周、1 个月、3 个月，或全部时间。
- **分类**：全部分类，或按类型筛选；在「休闲游戏检测」下，具体分类由右侧边栏的「新游戏 / 新玩法 / 竞品」及平台、公司决定。
- **排序**：默认排序、最新发布、最受欢迎、互动最多、评分最高、评分最低。

点击「高级筛选」可以展开更多选项，例如按情感（正面/中性/负面）、趋势方向（上升/稳定/下降）筛选。  
列表上方会显示「共找到 N 条监测数据」。

### 4. 查看详情

列表里每条监测以卡片形式展示。**有完整内容的条目**（如周报、日报）可以点击整张卡片，进入**详情页**阅读全文。

详情页会显示：

- 标题、来源、日期、时间、评分（如有）、标签
- 封面图（如有）
- 正文内容（支持标题、列表等格式）

左上角有 **「返回」** 按钮，点后回到列表，且会保留您之前选的类型、公司等条件。

### 5. 休闲游戏排行榜

在「休闲游戏检测 → 新游戏」下，点击 **「进入排行榜」**，会进入排行榜页面。

- 顶部有**平台标签**：微信小游戏、抖音小游戏、安卓游戏、iOS 游戏，可切换查看不同平台的周榜。
- 表格中展示：排名、游戏名称、开发商、分类、排名变化（上升/下降/新进榜）、更新时间等；排名变化会用不同颜色和箭头区分。
- 左上角有 **「返回」**，可回到休闲游戏列表。

### 6. 其他功能入口

- **搜索**：顶部有搜索框，可输入关键词搜索监测内容（支持快捷键 K）。
- **通知**：顶部有通知图标（有红点表示有新通知）。
- **深色/浅色**：可切换页面亮色或暗色显示。
- **登录**：顶部有登录入口，用于后续账号相关功能。

以上搜索、通知、深色模式、登录等为界面预留，具体以网站实际开放为准。

## 三、页面上有哪些内容（界面说明）

### 1. 整体布局

- **最上方**：一条固定的导航栏，始终可见。
- **中间偏左**：主内容区，用来显示「列表」「某条详情」或「游戏排行榜」中的一种。
- **右侧**：固定宽度的侧边栏，标题为「监测源」，用于选择类型、平台、公司等。

### 2. 顶部导航栏

- **左侧**：网站 Logo 和「监测汇总平台」名称。
- **中间**：四个 Tab——监测汇总、AI热点、趋势检测、休闲游戏检测；当前选中的会高亮（下划线、加粗）。
- **右侧**：搜索框、通知图标、深色/浅色切换、登录按钮。

### 3. 主内容区会出现的三种画面

**（1）列表页**

- 最上方是当前页标题（例如「监测汇总」或「休闲游戏检测 - 新游戏 - 微信」），在休闲游戏新游戏下右侧会有「进入排行榜」按钮。
- 下面是筛选栏：时间范围、分类、排序、高级筛选。
- 再往下是「共找到 N 条监测数据」。
- 列表由多张**监测卡片**组成，每张代表一条监测。

**（2）监测卡片长什么样**

- 左侧：封面图，或按类型显示不同颜色的色块和类型名称（如 AI热点检测、休闲游戏检测）；若有趋势，会在角上显示上升或下降。
- 右侧：标题、来源、平台、日期时间、浏览量、互动数、评分（如有）、简短描述、情感标签（正面/负面/中性）、若干标签。
- 整张卡片可点击：有详情的会打开详情页，没有详情的则不会跳转。

**（3）详情页**

- 顶部一条栏：返回按钮、文档标题、来源、日期、时间、评分、标签。
- 下面：可选的一张封面图，再下面是正文区域，支持标题、段落、列表等排版。

**（4）排行榜页**

- 顶部：排行榜标题和简短说明；从休闲游戏入口进入时会有「返回」按钮。
- 中间：平台标签（微信小游戏 / 抖音小游戏 / 安卓 / iOS），点击切换。
- 下方：大表格，列包括排名、游戏名、开发商、分类、排名变化、更新时间等，排名变化用颜色和箭头区分。

### 4. 右侧「监测源」边栏

- 最上方标题：「监测源」。
- **「全部」**：点选后，主区域显示所有类型的监测内容。
- **按类型**：下面会列出「AI 热点检测」「热点趋势检测」「休闲游戏检测」等，每类可带数量；当前选中的类型会蓝底高亮。
- **休闲游戏检测** 点开后会出现：
  - 新游戏、新玩法、竞品；
  - 选「新游戏」时，下面会列出平台：微信、抖音、iOS、安卓（周榜）；
  - 选「竞品」时，会列出：社媒更新、UA 素材；
  - 选「社媒更新」时，会列出「全部公司」以及各公司名称，可点选某公司只看该公司周报。
- 当前选中的类型、平台或公司都会用蓝底高亮显示。

### 5. 视觉与使用体验

- 页面以白色为底，主要按钮和选中状态为蓝色，卡片和按钮为圆角；鼠标悬停列表卡片时会有背景变化。
- 不同类型配有简单图标（如 AI、趋势、休闲游戏等），便于一眼区分。
- 在不同屏幕尺寸下，布局会自动调整，小屏幕时导航可能会收拢，便于在手机或平板上使用。

## 四、简要总结

| 项目 | 说明 |
|------|------|
| **网站用途** | 集中展示 AI 热点、热点趋势、竞品社媒、休闲游戏等多类监测内容，支持按类型、平台、公司筛选和排序。 |
| **您能做的事** | 切换类型看列表、点进某条看详情、在休闲游戏下按平台看周榜或进入排行榜看各平台游戏排名。 |
| **界面结构** | 顶栏选类型 → 右侧选监测源/平台/公司 → 中间看列表或详情或排行榜；列表以卡片形式展示，详情页统一展示标题、来源、时间、正文等。 |
